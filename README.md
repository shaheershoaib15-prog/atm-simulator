# C++ ATM Console Application

A simple and interactive ATM simulation program written in C++. This console-based application allows users to perform basic banking operations such as checking balance, depositing money, withdrawing cash, and transferring funds between accounts.

## Features
- Check account balance
- Deposit money into account
- Withdraw money from account with sufficient balance check
- Transfer funds between accounts
- Simple menu-driven interface
- Input validation for safe transactions

## How to Run
